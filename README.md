**Credit Card Analytics Dashboard**

**Overview**

This project analyzes credit card customer data and transaction patterns using Power BI, SQL, and Excel. It provides insights into customer behavior, revenue generation, and spending trends through interactive dashboards.

**Tools and Technologies**

- Microsoft Excel
- Power BI
- SQL

**Key Metrics**

1. Total Revenue: 55M
2. Total Interest: 7.84M
3. Total Transaction Amount: 45M
4. Total Transactions: 656K

**Customer Report Insights**

- Business and high-income customers contribute the most revenue
- Age group 50–60 shows highest revenue contribution
- Blue card category dominates usage
- Top regions include TX, NY, and CA
- Transaction Report Insights
- Bills and entertainment are the highest spending categories
- Swipe transactions are the most used mode
- Blue card category generates the majority of revenue
- Revenue remains consistent across quarters

**Features**

- Data cleaning and preprocessing in Excel
- Data extraction and transformation using SQL
- Interactive dashboards built in Power BI
- Customer segmentation and trend analysis
